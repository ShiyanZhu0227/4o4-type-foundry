Cirkit

Info

Cirkit is a variable font built around playful geometry and elastic motion. Its defining features are a stretchable stem axis and a circle module that seamlessly shift between solid and hollow states. These variation axes allow Cirkit to morph in rhythm with your layout, expanding, contracting, or opening up to create dynamic typographic expressions.
